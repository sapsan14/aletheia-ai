# Aletheia AI

**Verifiable AI responses** — PoC for cryptographically signed and timestamped LLM answers so that responses can be proven, not just trusted.

- **Architecture:** [docs/PoC.ru.md](docs/PoC.ru.md) · [docs/PoC.et.md](docs/PoC.et.md)
- **Implementation plan:** [docs/plan.en.md](docs/plan.en.md) · [docs/plan.ru.md](docs/plan.ru.md) · [docs/plan.et.md](docs/plan.et.md)
- **Diagrams:** [diagrams/architecture.md](diagrams/architecture.md)

Stack (PoC): Next.js, Java Spring Boot, PostgreSQL, OpenSSL/BouncyCastle, RFC 3161 TSA, one LLM (OpenAI/Gemini/Mistral).
